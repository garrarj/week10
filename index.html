CSS Transitions

Transitions allows you change property values smoothly, rather than instantaneously. It is important to remember that you cannot change between binary values, like between display: block; and display: none;, because there are not interim values between those two states. However, if there are values between the two states you wish to transition between, like between font-size: 10px; and font-size: 24px;, then CSS transitions can be applied.

Lottie

Developed by AirBNB, Lottie is a library that parses After Effects animations into json objects which can then be render natively on mobile platforms and the web. It's awesome.

  • Lottie documentation
  • Lottie Files is a site that has many free Lottie animations that you can use in your projects.

Lottie requires a few elements to function, but it's all fairly simple:

  1. An animation exported as a json object - In this week's example I have created an animation and saved it in the 'data' folder
  2. The lottie player - this can be loaded in your project files (as I have done here), or you can link to a CDN
  3. A simple javascript function that defines which HTML element will hold the animation, how the animation is rendered, whether it autoplays, etc.
  4. Hosting space - the json data will not be loaded if you try to run your website locally. In order to test and use this tool, you will need to upload your code to Github Pages (or other hosting service)
